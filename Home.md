## 🔍 Quick Reference

### Core Process Cycles

```
P2P:  PR → Approval → PO → GR → Invoice → Payment
S2C:  Spend Analysis → RFx → Bids → Award → Contract
RFx:  Create → Invite Suppliers → Publish → Bids → Compare → Award
```

### Key SAP Objects

| Object | Code | Purpose |
|---|---|---|
| Purchase Requisition | PR | Internal request to buy |
| Purchase Order | PO | Legal document to vendor |
| Goods Receipt | GR | Confirmation of delivery |
| Purchasing Info Record | PIR | Vendor-material pricing link |
| Outline Agreement | OA | Contract or scheduling agreement |
